# 🌍 WanderLust

**WanderLust** is a full-stack travel listing web application inspired by Airbnb. It allows users to explore various travel destinations, view detailed listings, and manage them via a simple UI. The app is built using Node.js, Express, MongoDB, and EJS templating.

---

## 🚀 Features

- 🏡 View all available travel listings
- ➕ Add a new listing with title, description, and price
- ✏️ Edit existing listings
- ❌ Delete a listing
- 📦 Server-side rendered using EJS templates
- 🎨 Responsive UI with Bootstrap styling

---

## 🛠️ Tech Stack

- **Frontend**: HTML, CSS, Bootstrap, EJS
- **Backend**: Node.js, Express
- **Database**: MongoDB, Mongoose
- **Templating**: EJS
- **Utilities**: Body-parser, Method-override
